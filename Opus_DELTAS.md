# Opus_DELTAS.md

Deviation and decision log for the Opus CC RAW mirror.

Per the Autonomy Charter, this file records:

- any deviation from an architecture doc,
- any implementation decision where the doc was silent,
- any component-level pattern chosen,
- any provider behavior observed that differs from documented limits.

One row per decision. Newest entries at the bottom of each phase block.

| 2026-06-14 | Dispatcher | Deviation | A provider 4xx now FAILS OVER to the next provider instead of aborting the dispatch. Previously a `RequestError` (4xx) was re-thrown out of the failover loop ("no retry, no failover, surface it"), which in a live multi-provider run let one provider's 400 (e.g. a model id its endpoint rejects) kill the whole turn and PAUSE the orchestrator, even when another keyed provider would succeed.
